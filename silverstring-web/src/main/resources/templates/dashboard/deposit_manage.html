회원님의 입금 주소 :


을 판매하기 위해서는 먼저 회원님의 Coinest 전자지갑에 판매할 을 직접 입금하시거나 다른 회원으로부터 을 구매하셔야 합니다.
회원님께 할당된 위 코인 주소로 을 전송하시면 회원님의 전자지갑에 반영됩니다.
반영 시점: 입금거래후 10~30분 이후 자동으로 반영됩니다.
